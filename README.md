This project is an implementation of the famous Push swap exercise, completed as part of the 42 curriculum.

## Description

Push swap is a program that sorts a stack of integers using a limited set of basic operations (`push`, `swap`, `rotate`, `reverse rotate`). The goal is to find the most efficient algorithm to sort the stack in as few moves as possible.

## Algorithm Used

The main algorithm used here is a **custom and optimized chunk sort**.  
It divides the stack into multiple subsets ("chunks") to optimize sorting, especially for large lists.

**Details:**  
The optimization of sorting once the chunks are prepared takes a lot of computation time, as the program systematically seeks the most optimal moves for each element to be processed.  
This helps reduce the total number of operations but significantly increases computation time as the stack size grows.

**Advantages:**
- Allows for a reduced number of operations
- Suitable for medium to large lists

**Limitation:**
- Increasing computation time with stack size, due to searching for the most efficient moves for each chunk

## Benchmarks and Visualization

Benchmarks have been performed to measure the performance and efficiency of the algorithm on different stack sizes.

To measure the program's performance, SimonCROS's [push_swap_tester](https://github.com/SimonCROS/push_swap_tester) was used.

To better understand how sorting happens and the moves performed, the project was also tested with [Push swap visualizer](https://github.com/o-reo/push_swap_visualizer).  
This tool allows real-time visualization of operations and helps verify the consistency of the sorting generated by the algorithm.
